# V0 Baseline Evidence

## Baseline Position

V0 is the intentionally rough starting point of the maturity demo.

## What V0 Includes

- A working daily activity journal website
- Basic local storage entry flow
- Deliberately clunky UI and weak refinement
- No structured repository AI architecture layers

## What V0 Excludes

- No `.github/copilot-instructions.md`
- No reusable prompt files
- No focused agent definitions
- No skill definitions
- No MCP wiring
- No hook-based validation gates

## Supposed Prompt Provenance

V0 is treated as produced by one broad, low-quality one-shot prompt.

Reference prompt record: [../V0_BAD_PROMPT.md](../V0_BAD_PROMPT.md)

## Why This Baseline Is Useful

This provides a stable point for demonstrating quality deltas introduced by each subsequent architecture layer.
