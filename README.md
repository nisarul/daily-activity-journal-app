# Daily Activity Journal

A deliberately rough V0 website for logging daily activities.

## Run

```bash
npm start
```

Open `http://localhost:3000`.

## What it does

- Save daily journal entries
- Track mood and notes
- Show a simple entry list and quick stats

## V0 traits

- Plain, cluttered layout
- Default controls and awkward styling
- Working behavior first, polish later
